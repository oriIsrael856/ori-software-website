# תוכנית עבודה — אתר תדמית לעסק פתרונות תוכנה

> מטרת-על: אתר שמביא **לקוחות**. כל החלטה עיצובית/טכנית נמדדת מול השאלה —
> "האם זה גורם ללקוח פוטנציאלי לסמוך עליי וליצור קשר?"

## שלב 0 — יסודות ותוכן (לפני שורת קוד אחת)

- [ ] **מיצוב**: משפט אחד שמסביר מה אתה עושה ולמי (למשל: "פתרונות תוכנה בהתאמה אישית לעסקים — מרעיון למוצר עובד").
- [ ] **קהל יעד**: בעלי עסקים קטנים-בינוניים? סטארטאפים? חברות? זה קובע טון, שפה ומחירון.
- [x] **בחירת 3–5 פרויקטים** להצגה — נבחרו 5: Stav Cosmetics, Gan Nuna Banuna,
  La Hilula, Jazz & Blues, Cursor RTL Patch. ממתין: אישורי לקוחות/שותף, צילומי מסך. לכל אחד: צילומי מסך/וידאו, הבעיה, הפתרון, טכנולוגיות, ותוצאה מדידה אם יש ("קיצר תהליך מ-3 שעות ל-5 דקות").
- [ ] **טקסט "עליי"**: קצר, אישי, ממוקד ערך ללקוח (לא קורות חיים).
- [ ] **ערוץ המרה ראשי**: וואטסאפ? טופס? קביעת שיחה (Calendly)? — מומלץ וואטסאפ + טופס.
- [ ] תמונת פרופיל איכותית, לוגו/שם עסק, דומיין (לרכוש מוקדם).

## שלב 1 — תשתית טכנית

- **סטאק מומלץ**: Next.js 15 (App Router) + TypeScript + Tailwind CSS 4 + Framer Motion.
  - Next.js — SEO מעולה (חובה בשביל לקוחות מגוגל), ביצועים, פריסה חינמית ב-Vercel.
  - Framer Motion — אנימציות ברמה הגבוהה שאתה רוצה.
- [x] `create-next-app` + הגדרת RTL מלא (`dir="rtl"`, `lang="he"`).
- [x] פונטים עבריים איכותיים (Heebo / Assistant / Noto Sans Hebrew דרך `next/font`).
- [ ] Git + GitHub repo + פריסה ל-Vercel כבר מהיום הראשון (כל push עולה לאוויר).

## שלב 2 — שפה עיצובית (Design System)

- [x] פלטת צבעים: רקע כהה עמוק + צבע מבטא אחד בולט (gradient) — משדר "טק" ופרימיום. תמיכה מלאה בניגודיות (WCAG AA).
- [x] סקאלת טיפוגרפיה דרמטית — כותרת הירו ענקית (clamp responsive).
- [x] טוקנים: ריווחים, רדיוסים, צללים, מהירויות אנימציה — קובץ אחד מרכזי.
- [x] קומפוננטות בסיס: כפתור (עם hover מיוחד), כרטיס, תגית טכנולוגיה, קונטיינר סקשן.

## שלב 3 — סקשן הירו (הרושם הראשוני)

- [x] כותרת ענקית עם אנימציית כניסה (טקסט שנחשף מילה-מילה / gradient נע).
- [x] רקע חי (גריד + glow נושם): gradient mesh מונפש או חלקיקים עדינים (Canvas/WebGL קליל) — מרשים אבל לא מסיח.
- [x] תת-כותרת של שורה אחת + שני CTA: "דברו איתי" (ראשי) ו"צפו בעבודות" (משני).
- [ ] אפקט עדין שמגיב לעכבר (parallax/tilt) — מדגים תשומת לב לפרטים.

## שלב 4 — גלריית פרויקטים (הלב של האתר)

- [x] אנימציות scroll-reveal (הכרטיסים נכנסים תוך כדי גלילה, stagger).
- [ ] עמוד/מודאל פרויקט מורחב: סיפור הפרויקט — בעיה → פתרון → תוצאה, גלריית מסכים, טכנולוגיות.
